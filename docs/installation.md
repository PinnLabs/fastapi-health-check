# Installation

## Install with uv

```bash
uv add fastapi-ht
```

## Install with pip

```bash
pip install fastapi-ht
```

## Requirements

- Python 3.12+
- FastAPI application

## What gets installed

The package includes:

- the health check registry
- the check base classes
- the function-based `health_check()` helper
- the built-in HTML UI renderer
- the FastAPI integration layer
